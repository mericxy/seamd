# Vision

## Overview

O objetivo do Seamd é ser um leitor de Markdown minimalista, leve e focado em desenvolvedores.

Ele se integra diretamente com projetos que utilizam arquivos `.md`, funcionando como uma ferramenta **read-first**, com opção de edição quando necessário.

## Problem

Editores de código tratam arquivos Markdown como código (edit-first), exigindo ações extras para visualização (preview, split view, etc).

Ferramentas como Notion não trabalham diretamente com arquivos `.md`, e outras soluções podem ser pesadas ou complexas.

## Solution

O Seamd propõe uma experiência:

- leitura como padrão (read-first)
- edição simples e rápida quando necessário
- integração direta com arquivos `.md` locais
- interface minimalista e sem distrações

## Target Audience

- Desenvolvedores
- Pessoas que usam Markdown para documentação
- Usuários que preferem arquivos locais ao invés de ferramentas cloud

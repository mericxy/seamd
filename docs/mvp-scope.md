# MVP

## Core Features

- abrir arquivo `.md`
- leitura bonita por padrão (read-first)
- botão/atalho para editar
- salvar no mesmo arquivo

## File Navigation

- sidebar com arquivos da pasta
- foco em arquivos `.md`
- navegação por diretórios

## Markdown Rendering

Suporte para:

- headings
- listas
- tabelas
- code blocks
- links
- imagens locais

## UX

- busca no documento
- tema claro/escuro

## Non-goals (por enquanto)

- sincronização em nuvem
- plugins
- colaboração em tempo real
- suporte completo a todos os padrões Markdown
